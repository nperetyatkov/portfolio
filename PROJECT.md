# PROJECT — Portfolio

## 1. Назначение
Персональный сайт-портфолио UX/UI-дизайнера Николая Перетяткова на чистом HTML, CSS и JavaScript.

## 2. Текущий статус
- Разработку сайта можно считать завершённой.
- Production-версия опубликована через GitHub Pages.
- Основной домен: `https://nperetyatkov.ru`
- Основная ветка проекта: `main`

Состав production-сайта:
- главная страница: `/`
- кастомная страница 404: `/404.html`
- кейс `Multyne`: `/project-multyne/`
- кейс `mos.ru`: `/project-mosru/`
- кейс `START`: `/project-start/`
- кейс `UI Concepts`: `/project-ui-concepts/`

Подключено и используется в проде:
- адаптивная вёрстка главной страницы и страниц кейсов;
- единый шаблон страниц проектов;
- мобильное меню;
- плавный скролл по якорям с учётом фиксированного хедера;
- dropdown-навигация по секциям кейса с активным состоянием;
- единый JSON-источник пунктов навигации кейса через `case-nav-data`;
- кастомный курсор на desktop/tablet;
- авторасстановка неразрывных пробелов в текстовых блоках;
- просмотр изображений кейса в lightbox;
- аналитика: Google Analytics 4 и Яндекс.Метрика;
- локальный opt-out аналитики через параметр `?analytics=off` с сохранением состояния в `localStorage`;
- SEO-база: `CNAME`, `robots.txt`, `sitemap.xml`, canonical, Open Graph, Twitter meta и JSON-LD (`Person`, `WebSite`) для главной страницы.
- кастомная страница `404.html` для GitHub Pages.

## 3. Технологии
- HTML5
- CSS
- Vanilla JavaScript (ES modules)
- Git + GitHub Pages

## 5. Архитектурные принципы
- Один файл отвечает за одну зону ответственности.
- CSS разделён по слоям: `base`, `layout`, `components`, `pages`.
- Общая инициализация интерактива выполняется через `scripts/main.js`.
- Переиспользуемая логика вынесена в `scripts/components/`.
- Страницы кейсов используют единый шаблон и общий набор стилей.

## 6. Поддержка и изменения
- `PROJECT.md` должен отражать текущее состояние production-версии.
- При изменении структуры проекта, страниц, домена, SEO-файлов или ключевого интерактива этот файл нужно обновлять.
- `sitemap.xml`, `robots.txt` и метаданные страниц должны оставаться синхронизированными с доменом `https://nperetyatkov.ru`.
- Для веса шрифта использовать токены: `--fw-regular`, `--fw-medium`, `--fw-semibold`, `--fw-bold`.

## 7. Визуальные ориентиры
- минималистичная подача;
- акцент на типографику, отступы и визуальную иерархию;
- аккуратные анимации без перегрузки интерфейса.
